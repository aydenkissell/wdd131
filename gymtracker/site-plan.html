Gym Progress Tracker - Site Plan

Purpose

This website helps users track their workouts, weights, and progress over time. Users can log exercises and view improvements, helping them stay consistent and motivated in the gym.

Audience

This site is designed for gym-goers, especially beginners and college students who want a simple way to track workouts without complicated apps. It is ideal for people focused on building muscle, losing weight, or improving consistency.

Dynamic Elements

  • Users can add workouts using input fields (exercise, weight, reps).
  • Workouts will display dynamically on the page using JavaScript.
  • Data will be stored using localStorage.
  • Users can filter workouts by muscle group using array methods.
  • Conditional logic will be used to display messages (e.g., strong lift vs keep improving).
